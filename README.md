# Whacki — Whack-A-Mole Game 🎯

A fun browser-based Whack-A-Mole game built with JavaScript, HTML, and CSS.

## 🚀 Live Demo
👉 [Play Here](https://whacki.netlify.app)

---

## ✨ Features
- 🎮 Multiple mole types (normal, king, bomb)  
- 📊 Score tracking & time countdown  
- 🎯 Randomized mole appearance  
- 🖱️ Click/tap detection for whacking  
- 🔊 Sound effects for hits & bombs  

---

## 🕹️ How to Play
1. Click **Start** to begin the game.  
2. Moles will pop up randomly from holes.  
3. **Click a mole** to earn points.  
4. ⚠️ Avoid bombs or you’ll lose points.  
5. The game ends when the timer runs out.  

---

## 🛠️ Built With
- **JavaScript (Vanilla)** — game logic & event handling  
- **HTML5** — structure  
- **CSS3** — styling & animations
